# 高级视觉排版规则

## 目标

精美排版不是把原文档截图贴进 HTML，而是把已识别内容重新组织成“设计师排过”的纸面资料。

最终交付必须包含两层：

1. 重排正文：可复制、可编辑、可打印的 HTML 内容，是主交付。
2. 原卷校验附录：逐页图片或题内原图，只用于核对题量、图形和公式，不得替代正文。

如果当前链路无法 OCR 或结构化出完整题目，必须停止说明缺少 OCR/题目结构化结果，不能用整页图片冒充排版。

## 高级感来源

高级感来自信息秩序，不来自花哨装饰：

- 统一网格：稳定页边距、题区间距、题号宽度、答题区长度。
- 清晰层级：标题、考试信息、题型标题、题干、选项、答题区视觉层级分明。
- 高信息密度：充分利用 A4 纸面，但保留呼吸感。
- 克制配色：默认黑白灰加一枚低饱和强调色，避免大面积彩底。
- 精准对齐：题号、选项、横线、表格、分栏边界必须对齐。
- 可打印优先：白底、深色文字、细线框、浅灰底纹，不依赖屏幕特效。

## 视觉规范

- 页面：A4 纵向，`12mm 14mm` 到 `14mm 16mm` 页边距。
- 字体：中文优先 `Noto Serif SC`, `Songti SC`, `SimSun`；题干可用衬线，辅助信息可用无衬线。
- 主标题：22-28px，字重 700，避免过大。
- 题型标题：用细线、浅底或编号徽标区分，不使用大色块。
- 题目：题号固定宽度，题干和选项左对齐。
- 选项：数学选择题优先 2 栏或 4 栏网格，短选项不得散乱。
- 填空：使用真实边框线或 `border-bottom`，不能用一串下划线字符。
- 表格：细线边框、表头浅灰底，禁止重边框。
- 公式：MathJax 渲染，行内公式不破坏行高，展示公式保持居中。
- 图形：能提取题内原图时放回题目附近；不能提取时，在附录保留整页原图并在正文标注“见附录第 n 页原图核对”，但正文不能因此缺题。
- 图形题：如果无法提取题内图，不能写“请参考原卷图示”来糊弄；要么回填真实题内图，要么停止说明缺少题内图片裁剪/定位能力。
- 统计图题：必须保留图表标题、数据、子问和判断项，不能简写成“统计图分析”。

## 禁止的低级排版

- 正文只有整页原卷图片。
- 只有图片，没有可编辑题干、题号、题型标题。
- 用“原卷保真层”代替重排正文。
- 用“请参考原卷图示”“统计图分析”“补充统计图”等摘要替代图形题正文。
- 把一道含多个子问的大题压成一句概括。
- 大面积渐变、黑底白字、花哨贴纸、无意义英文装饰。
- 题目卡片间距过大，导致 A4 纸张浪费。
- 每题都加厚重边框，视觉噪声过强。
- 题号、选项、横线不对齐。

## 验收

- [ ] 正文主体是可复制 HTML，不是整页图片。
- [ ] HTML 中有明确题型结构和题号结构。
- [ ] 原卷图片只在附录或题内图位置出现，不充当整页正文。
- [ ] 图形题、统计图题没有被摘要或“参考原卷”替代。
- [ ] 样式高级克制，适合打印和分发。
- [ ] 纸面信息密度合理，没有大面积空白或低幼装饰。
