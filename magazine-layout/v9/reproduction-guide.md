# 内容复现与模板锁定规则

## 目标

精美排版的第一步不是美化，而是把老师给定的内容完整、准确地落到 HTML 中。只有内容复现通过，才能进入排版。

## 内容清点

处理上传文档或粘贴内容时，先建立清点清单：

| 类型 | 必须清点 |
|---|---|
| 试卷/题单 | 标题、题型顺序、每类题目数量、题号、选项、答案区、图表 |
| 英语练习 | Unit、词汇、语法点、阅读文本、题型、填空横线、写作区 |
| 数学/物理 | 公式、分数、单位、已知量、图形、电路图、光路图、题号 |
| 语文 | 原文、古诗文、注释、题干、阅读材料、题型、答题区 |
| 教案 | 教学目标、重难点、教学环节、活动顺序、板书/作业 |

## 零散题目整理

老师上传的是零散问题时，先做“收拢”，不是“创作”：

- 原题全部保留。
- 根据题目原有标识或题干特征分组：选择题、填空题、判断题、解答题、阅读题、写作题。
- 没有题号时重新编号；已有题号时沿用原题号。
- 题干和选项不能改写，只能修正明显的排版空格。
- 答题区可按题型补充，但不得改变题目要求。

如果输入只有“生成某年级某知识点若干题”，但没有具体题目内容，这不是本 Skill 的“已给定内容排版”场景。应先由组题/命题能力生成内容，再把生成结果交给本 Skill 排版。

## 强制复现规则

- 上传文档时，原文档是第一权威来源。
- 题目数量必须一致；无法完整呈现时拆页，不删题、不合并题、不只保留前几题。
- 原题不得替换、改写、合并或摘要化。
- 原始标题必须保留，除非用户明确要求重命名。
- 原图优先保留；学科图不能随意重绘，不能因为提取困难就丢弃。
- 公式、分数、单位、已知量必须逐项核对；LaTeX 分数必须渲染为数学公式，不能显示源码。
- 教案必须保留教学环节，不改造成阅读材料。

## 试卷/PDF/截图专项规则

- 如果原卷是截图或扫描 PDF，先把每页作为完整视觉资产对待。
- 能可靠提取文字和图片时，文字重排、图片回填到对应题目。
- 图片无法可靠裁剪时，必须保留原页面截图或原图区域作为兜底，不能直接省略。
- 只有 PDF URL、没有逐页图片或截图 URL 时，不能用空白 PDF 框冒充保真；必须说明需要 PDF 转图能力。
- 只有 resourceId 或 PDF URL 时，不能按命名规律猜测图片地址，必须使用工具真实返回的图片 URL。
- 没有真实图片 URL 时，不能生成 HTML；必须停止并说明缺少 PDF 转图/图片提取能力。
- 如果 OCR/解析只拿到部分题目，不能用部分题目交付“精美排版”；应保留原页图或提示内容不完整。
- 题量核对必须按题型做：例如选择题 20 题、填空题 12 分、解答题 6 题。不能只生成前 7 题。

## 不能做的事

- 不因为排版好看而删减内容。
- 不用“精选题目”替代“全部题目”。
- 不把上传文档改写成新资料。
- 不把原卷里的几何图、表格、图形题删除。
- 不写“[此处原卷为：...]”来代替原图。
- 不写“更多题目请参考原卷”“请参考原卷保真层”来代替未复现的题目。
- 不用 `<object>`、`<embed>`、`<iframe>` 空白 PDF 框代替逐页图片。
- 不编造 `_page_1.png`、`pdf?page=1` 等图片地址。
- 不用 `onerror` 隐藏图片加载失败。
- 不把 `\(\dfrac{7}{9}\)` 这类分数源码原样显示在卷面上。
- 不在中文资料中加入无意义英文标题。
- 不默认黑底白字。
- 不默认双栏试卷，除非原文档或用户要求。

## 模板锁定

当用户说以下任一句，触发模板锁定：

- “记住这个格式”
- “下次按这个来”
- “按照 resourceId XXXX 一模一样的格式”
- “参考我给的文件，格式排版要一模一样”
- “继续制作 Unit 5”
- “下一个单元也按这个模板生成”

### 固定层

锁定后不得自行改变：

- 布局方式：紧凑流式、单栏、双栏、分栏规则
- 配色方案
- 字体字号
- 页边距
- 题型结构顺序
- 排版规则：如短语辨析 Grid 每行 5 个、填空横线长度
- 技术规格：纯 HTML + CSS + A4 打印适配

### 可变层

每次可替换：

- 年级、学科、单元
- 具体词汇和语法内容
- 阅读语篇文本
- 题目内容
- 标题中的单元名/期次

## 自检

进入排版前确认：

- [ ] 没有删题、换题、漏题。
- [ ] 没有遗漏原图、公式、表格。
- [ ] 题型顺序和题号连续。
- [ ] 场景类型已判断：试卷、题单、讲义、教案、默写纸、知识清单、模板复用。
- [ ] 模板锁定时固定层没有漂移。
