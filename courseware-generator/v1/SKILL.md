# 多页课件生成技能

本技能指导 AI 完成多页 PPT 式教学课件的完整生成流程，从内容规划到 HTML 输出。

---

## 技能文件说明

本技能由三个文件组成，请按需读取：

| 文件 | 用途 | 何时读取 |
|------|------|----------|
| **SKILL.md**（本文件） | 总览与导航，说明整体流程和文件职责 | 首先读取 |
| **[content-guide.md](content-guide.md)** | 课件**内容**生成指南：确认参数→检索资料→规划大纲→分批生成各页教学内容→验收组装 | 开始制作课件时读取，贯穿全流程 |
| **[style-guide.md](style-guide.md)** | 课件 **HTML 格式与样式**规范：HTML 文件结构、`<template>` 标签用法、960×540 画布规则、壳框架约束、互动状态管理、禁止事项等 | 编写页面 HTML 代码、组装最终文件时读取 |

---

## 工作流程

```
1. 读取本文件 → 了解整体流程和文件职责
     ↓
2. 读取 content-guide.md → 按其工作流执行：
   确认参数 → 检索原文与教案 → 规划大纲 → 分批生成各页内容
     ↓
3. 读取 style-guide.md → 按其 HTML 结构规范和样式要求：
   将各页内容编写为符合壳框架要求的 <template> 标签
     ↓
4. 验收与组装 → 按 content-guide.md 的验收流程核对
   → 一次性组装为最终 HTML 文件
```

---

## 核心原则

- **内容先行**：先按 content-guide.md 完成内容规划和大纲设计，再按 style-guide.md 编写 HTML。
- **分批生成**：不一次性生成全部页面，按批次生成片段文件，最后一次性组装。
- **壳框架驱动**：AI 只负责编写 `<template>` 内的教学内容，壳功能（缩略图、演示模式、翻页等）由云端 JS 自动提供，**禁止手写壳代码**。
- **最终交付一个 `.html` 文件**：浏览器打开即可使用，无需其他文件或服务器。

---

## 使用说明

1. **收到课件生成需求后**，依次读取 content-guide.md 和 style-guide.md。
2. content-guide.md 负责「做什么」——规划课件的教学内容、结构、每页大纲。
3. style-guide.md 负责「怎么做」——将内容转化为符合壳框架规范的 HTML 代码。
4. 两份指南须**同时遵守**，内容指南决定每页写什么，样式指南决定怎么写成 HTML。
